Create four tables as shown in the following ER diagram. You must include the primary keys, foreign keys and some suitable data types. All columns should be non-nullable.

Table: vehicle

Columns:
  • vehicle_regn_number (Primary Key)
  • registration_date
  • value

Table: driver

Columns:
  • driver_number (Primary Key)
  • first_name
  • last_name

Table: insurance_policy

Columns:
  • policy_number (Primary Key)
  • vehicle_regn_number (Foreign Key referencing vehicle table)
  • start_date
  • end_date
  • driver_number (Foreign Key referencing driver table)

Table: claim

Columns:
  • claim_number (Primary Key)
  • policy_number (Foreign Key referencing insurance_policy table)
  • driver_number (Foreign Key referencing driver table)
  • claim_amount
