Universal Selector

The CSS universal selector selects all the elements on the webpage. It is used when we want to specify the same CSS property to every element on a web page.

Let's understand it better with the help of an example, suppose, we want to specify a specific font type instead of the browser's default font or want to set the background color for a webpage, then, with the help of a universal selector, we can specify any CSS property universally (for all the elements on the web page).

The universal selector can be specified using the asterisk sign (*).
