Introduction to Selectors

CSS selectors are used to targeting the specific HTML elements on the webpage for which we want to set certain styling properties. With the help of selectors, we can also set style rules universally which means the specified CSS properties will then be applicable to all the elements present on the webpage. Also, we can set styling using tag name, class, id, etc.

  • Universal Selector
  • Element Selector
  • Class Selector
  • Id Selector
