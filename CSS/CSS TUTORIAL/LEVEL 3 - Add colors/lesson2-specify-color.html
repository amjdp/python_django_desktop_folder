Specifying color using HEX value

The HEX values or Hexa code are the most used color values. These values are mostly used by web designers and professionals.

The HEX value starts with a hash (#) symbol and then followed by three or six characters, where using three characters is a shorthand form and six characters is the long-form.

The syntax of the Hex value is #RRGGBB,where RR = Red GG = Green BB = Blue

The hex value consists of the numbers as well as the characters because the hexadecimal number system contains 16 digits where 0 to 9 which is 10 in total are digits. The remaining 6 digits are represented by the characters A to F.

Here is the list of some common colors with their hex values below:

Color Name Hex value
White #FFFFFF
Black #000000
Red #FF0000
Yellow #FFFF00
Green #008000
Purple #800080
Teal #008080
Aqua #00FFFF
Gray #808080
Blue #0000FF

Specifying color using RGB

The RGB stands for red, green, and blue and can be specified using the rgb() function.

The syntax of the RGB is rgb(red, green, blue) where the value of red, blue, and green lies between 0 to 255.

Generally, the RGB format is the least used if you just want to use a common color value. But if you want, you can easily find different tools online using which you can easily convert a color value to HEX code and RGB values.

p{ color: rgb(239, 79, 79); }
